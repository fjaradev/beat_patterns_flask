# Beat Patterns Sequencer

A web application designed for rhythm training and practice. This app generates and plays randomized 4-bar rhythm patterns to help users improve their timing and rhythmic reading skills.

## Features

- **Randomized 4-Bar Patterns**: Generates 4-bar patterns based on the subdivisions you choose.
- **Adjustable Settings**: Go to the settings page to select which subdivisions you want included in your generated patterns. (More settings coming in future releases!)
- **Playback Controls**: Play and stop the generated rhythm pattern.
- **Looping**: Toggle the repeat loop on or off with the loop button to continuously practice the same pattern or play it just once.

## Tech Stack

This project is built using:
- **Backend**: Python with [FastAPI](https://fastapi.tiangolo.com/)
- **Frontend**: HTML, CSS, JavaScript (served via Jinja2 templates)
